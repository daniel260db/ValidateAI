# ValidateAI

ValidateAI is a web app that helps founders and builders quickly evaluate business ideas using structured AI analysis.

## What it does
- Scores ideas in seconds
- Highlights risks, costs, and complexity
- Gives a clear BUILD / DON'T BUILD verdict

## Tech stack
- Next.js (App Router)
- Supabase (auth + database)
- OpenAI (idea scoring)
- Stripe (subscriptions)
